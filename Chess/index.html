# White Black

It is I, Vamidus ⧉, who built this little chess game as a fun project to practice my web development skills. I handled the front-end, making the UI work and ensuring it looks decent on your screen. You might notice the computer player is pretty smart; that's thanks to an external chess engine I integrated.

It's powered by the "p4wn" engine under the hood, which handles all the deep thinking for the computer's moves. I didn't write that part myself; I simply integrated this awesome open-source technology so you have a challenging opponent to play against!

I hope you have a great time playing a few games. If you're interested in checking out the tech that makes the opponent tick, check out the p4wn engine GitHub page ⧉ – the folks who built it are the real MVPs behind the AI. Enjoy the game!
